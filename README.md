## 📋 Описание

Полный, актуальный список всех рабочих промокодов для игры **Black Russia** (Блэк Раша) — многопользовательской мобильной игры на Android, созданной на базе "GTA Криминальная Россия".

Этот сайт собирает все известные промокоды, включая:
- ✅ Основные промокоды
- ✅ Промокоды от стримеров
- ✅ Промокоды по серверам (RED, GREEN, BLUE, YELLOW и другие)
- ✅ Информацию о награды и статусе каждого кода

## 💡 Как использовать коды

1. **Откройте игру Black Russia**
2. **Нажмите на иконку "Меню"** (три полоски в правой части экрана)
3. **Перейдите в раздел "Промокод"**
4. **Введите промокод** из этого списка
5. **Получите награду!**

⚠️ **Важно:** Вводить можно только ОДИН промокод от ютубера за раз! При попытке ввести второй появится ошибка.
